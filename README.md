### 标准文件结构

	[mainfolder]
	 |--[js]		//js文件夹
	 |   |-- main.js
	 |--[style]		//所有样式相关的css和image
	 |   |-- [image]	//主要image文件夹
	 |   |    |-- img1.png
	 |   |    ...
	 |   |-- [styleNameA]	//皮肤A的文件夹
	 |   |    |-- [image]	//皮肤A的image
	 |   |    |-- styleNameA.css	//皮肤A的css文件
	 |   |    ...
	 |   |-- [styleNameB]	//皮肤B的文件夹
	 |   |-- [styleNameC]	//皮肤C的文件夹
	 |   |-- main.css	//主要css文件
 	 |-- index.html		//index文件
 	 |-- page1.html		//其他页面
 	 ...
	  
	  
ps：文件名全部英文小写，尽量不要加复数s，必要时可以使用下划线“_”。
